Novidades do mundo Linux

Canonical está desenvolvendo o Ubuntu 25.04 . Uma versão não LTS do sistema Ubuntu. Vamos conferir as novidades?

- O Ubuntu 25.04 está sendo desenvolvido com o kernel 5.17, que traz melhorias de desempenho e segurança para o sistema.

- O sistema também terá suporte para o Wayland, que é um servidor gráfico que substitui o Xorg e traz melhorias de desempenho e segurança para o sistema.

- O Ubuntu 25.04 também terá suporte para o Flatpak, que é um sistema de empacotamento de aplicativos que permite instalar aplicativos de forma segura e fácil.

O que é o Linux?

Linux não é como os outros sistemas operacionais que existem. Linux em sí é em modo texto e somente o kernel. As distribuições Linux quem colocam a interface gráfica e um pouco de customização no sistema.

- Um exemplo disso seria o Ubuntu, feito pela Canonical. Eles pegam o sistema base, no caso o kernel e customizam com a adição do Gnome ou KDE ou qualquer outra interface gráfica.

- O Android, feito pela Google e mantido pela Google, também seria uma distribuição Linux. O Android puro é de uma forma, como visto nos Google Pixel, mas outras empresas como a Samsung, customizam e tornam o sistema mais fácil e bonito.

A história do Linux

Linux é um sistema operacional de código aberto, baseado no kernel Linux, criado por Linus Torvalds em 1991. O sistema foi desenvolvido para ser uma alternativa ao sistema operacional Unix, que era muito caro e restrito. O Linux é um sistema operacional gratuito e de código aberto, o que significa que qualquer pessoa pode usá-lo, modificá-lo e distribuí-lo livremente. O Linux é usado em uma variedade de dispositivos, desde servidores de internet até smartphones e tablets. O sistema é conhecido por sua estabilidade, segurança e flexibilidade, e é amplamente utilizado em todo o mundo.
